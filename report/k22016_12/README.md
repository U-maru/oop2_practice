## 仕様

- flaskを使った画像処理クイズアプリ
- 画像をアップロードできる
- アップロードした画像の一覧が見れる
- （画像処理した画像の一覧が見れる）検討中
- アップロードした画像の中から好きな画像を選択してその画像でクイズを行う
- 正誤画面で単語の解説が見れる
- CSSで画面を見やすくする
- （アップロードした画像を消去）検討中
- 他の機能も随時追加
